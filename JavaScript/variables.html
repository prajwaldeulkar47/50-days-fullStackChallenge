Variables in JavaScript

In JavaScript, variables are used to store data values. You can declare a variable using the var, let, or const keywords.

Declaring Variables

// Using var
var name = "John";
// Using let
let age = 30;
// Using const
const country = "USA";

Variable Scope

The scope of a variable determines where it can be accessed in the code. Variables declared with var have function scope, while those declared with let and const have block scope.

Example

function example() {
    if (true) {
        var x = 10; // function scope
        let y = 20; // block scope
        const z = 30; // block scope
    }
    console.log(x); // 10
    console.log(y); // ReferenceError: y is not defined
    console.log(z); // ReferenceError: z is not defined
}
example();
